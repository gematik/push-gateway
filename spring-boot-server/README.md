# Push-Gateway: spring-boot-server

This module implements [A_27164](https://gemspec.gematik.de/docs/gemF/gemF_PushNotification/gemF_PushNotification_V1.1.0/#A_27164) by generating the REST interface from the `OpenAPI yaml file` in the `spec` module of this project. Please note that the actual implementation of the REST interface is done in the `backend` module of this project.

# Push-Gateway: spec

This module provides the `OpenAPI spec file` as stated in [A_27164](https://gemspec.gematik.de/docs/gemF/gemF_PushNotification/gemF_PushNotification_V1.1.0/#A_27164). Its purpose is to enable other modules to generate code from it which is implemented in the `spring-boot-server` module of this project.
